#  Simulador de Máquina de Turing 

Este sistema web simula el comportamiento de una máquina de Turing diseñada para analizar dos cadenas binarias separadas por un espacio. 
La herramienta permite seguir el recorrido de estados, observar el desplazamiento del cabezal y auditar cada instante de la cinta mediante el (DI).

---

## Descripción General

El simulador acepta una entrada estructurada como w ␣ x, donde w y x son secuencias compuestas por ceros y unos. 
Su propósito es determinar si ambas cadenas son idénticas, utilizando marcas específicas: A para los símbolos del primer bloque y B para los del segundo. 
La cinta se actualiza en tiempo real, mostrando el estado activo y el historial completo de transiciones.
